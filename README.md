# Data Cleaning & Preprocessing using Python

## Project Overview

This project demonstrates the complete Data Cleaning and Preprocessing workflow using Python. It prepares raw data for Machine Learning by handling missing values, encoding categorical data, scaling numerical features, detecting outliers, and saving the cleaned dataset.

---

## Objective

- Import a dataset
- Handle missing values
- Remove duplicate records
- Convert categorical data into numerical values
- Standardize numerical features
- Detect and visualize outliers
- Save the cleaned dataset
- Generate output files automatically

---

## Technologies Used

- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Installation

Clone the repository:

bash
git clone https://github.com/bobby0417/data-cleaning-project.git

Move into the project folder:

bash
cd data-cleaning-project

Install dependencies:

bash
pip install -r requirements.txt

---

## Run the Project

bash
python main.py
